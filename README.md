## 📖 Overview
**SportsDetector** is a Python tool that automatically finds **arbitrage betting opportunities** across multiple sportsbooks and notifies you instantly.

Arbitrage betting (aka **sure betting**) exploits differences in bookmaker odds to guarantee a profit regardless of the outcome. This project automates the process:

1) 📊 Scrape odds  
2) 🧮 Detect arbitrage opportunities  
3) 🔔 Send alerts (Telegram/SMS)  
4) ⏱ Repeat on an interval

---

## ✨ Features
- Real-time odds scraping  
- Arbitrage detection math  
- Telegram notifications (easy to extend)  
- Configurable scrape interval  
- Lightweight, extensible modules
